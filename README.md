# Local Roots — Capstone Project
**Protect & Support Your Community**

Local Roots is a community-based web app where neighbours can report cases of abuse or neglect, and request or offer everyday essentials like food, clothing, and shelter — all localised to their area.

---

### Pages
- **Homepage** — community stats and app overview
- **Report Page** — submit reports of child abuse, domestic violence, or elderly mistreatment
- **Help Board** — post or respond to needs (food, clothing, shelter)
- **Community Chat** — open local discussion

### Tech Stack
- React, Redux Toolkit, React Router
- Node.js, Express, MongoDB
- Deployed on Vercel (frontend) + Render (backend)
